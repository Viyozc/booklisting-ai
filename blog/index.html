KDP Keyword Research & Book Listing SEO That Gets You Found

Updated 2026-07-22 · BookListing team

Amazon's A9 is a search engine, not a library

Your book competes in a search box. When a reader searches "romantic suspense serial killer", Amazon's A9 algorithm ranks books by how well your metadata matches that query and how likely the click converts. Author rank helps, but relevance is king for new books. KDP keyword research is how you discover the exact phrases readers type — then place them where Amazon reads.

Category + keyword layering

Two levers move you: the Browse Categories (up to 2 on KDP, more via Author Central) and the 7 keyword fields (each up to 50 characters). Pick categories with demand but manageable competition, then reinforce them with keyword phrases that echo the category name. A cozy mystery set in a bakery should target "cozy mystery" and "baking mystery" as distinct fields, not repeat one phrase seven times.

Use all 7 keyword fields as phrases

Each of the 7 fields accepts a short phrase, not just one word. "small town romance", "enemies to lovers", "fake dating trope" — these multi-word fields match how readers actually search. Don't waste a field on a single generic word like "romance" when the phrase variant converts better and faces less competition.

The description is indexable too

Many authors treat the book description as sales copy only. Amazon does index it. Naturally include your top 2–3 keyword phrases in the first 150 characters and again lower down. Use HTML line breaks and bold to keep it scannable — a readable description lifts conversion, which lifts ranking.

Backend (hidden) keywords

The backend keyword field is where you capture spelling variants, translations, and phrases too long for the front 7 fields. Don't repeat words already in your title or categories — Amazon ignores duplicates. A free KDP listing audit flags missing coverage across all zones.

Common mistakes

  • Stuffing the title with keywords until it reads like spam.
  • Leaving keyword fields blank "because the category covers it".
  • Forgetting seasonal phrases (e.g., "stocking stuffer" in Q4).
